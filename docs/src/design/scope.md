# Scope

## Milestone 0

The path from source code to a rendered map, for a reduced feature set:

- crate boxes,
- dependency lines between crates within the project,
- the module tree inside each crate, nested to any depth.

The target is running the program on a Rust project, opening the website, and
seeing that project's crates and module trees.

## Excluded

- Functions, types, and other items inside modules.
- Compilation stages (source to running program).
- Links to GitHub.
- Learning paths.
- Anything else from the longer-term plan.
